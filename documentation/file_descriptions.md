# Описание файлов проекта

## Корневые файлы

*   `app.py`: Основной файл приложения Flask. Запускает веб-сервер, определяет маршруты (URL-адреса) и обрабатывает запросы.
*   `config.py`: Файл конфигурации. Содержит настройки приложения (порт, режим отладки, пути к файлам).
*   `requirements.txt`: Список Python-библиотек, необходимых для работы приложения.
*   `README.md`: Инструкция по установке, настройке и запуску приложения.

## Папка `backend/`

Содержит всю серверную логику приложения.

*   `__init__.py`: Делает папку `backend` Python-пакетом.
*   `models.py`: Определяет структуры данных (модели) для записей задач.
*   `processing.py`: Содержит функции для обработки данных (аналог `data_processing.py` из десктопной версии).
*   `storage.py`: Модуль для работы с хранилищем данных. Реализует чтение и запись в Excel файл (и в будущем в БД).
*   `utils.py`: Вспомогательные функции, общие для бэкенда.
*   `data/`: Папка для хранения данных.
    *   `Фотодня.xlsx`: Основной файл Excel, в который сохраняются и из которого загружаются записи.

## Папка `frontend/`

Содержит файлы пользовательского интерфейса.

*   `static/`: Папка для статических файлов.
    *   `css/style.css`: Файл стилей для оформления HTML-страниц.
    *   `js/script.js`: Файл JavaScript для добавления интерактивности на страницу.
*   `templates/`: Папка с HTML-шаблонами.
    *   `base.html`: Базовый шаблон, определяющий общую структуру страницы (заголовок, подключение стилей/скриптов).
    *   `index.html`: Главная страница приложения, содержащая форму ввода и таблицу последних задач.
